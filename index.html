Type vs. Class vs. ID

Specificity Syntax

CSS Code When Using Type, Class, & ID
Type <h1> <p> <html element>
Class .example
ID #example

Specificity Examples

Type Class ID
HTML <h2>This is a test</h2> <div class="test">Test Class</div> <div id="test">Test ID</div>
CSS .type {
color: green;
font-family: monospace;
}		 .test {
color: indianred;
font-weight: bold;
}		 #final {
color: maroon;
font-style: italic;
}
Output This is a Type Test This is a Class Test This is an ID Test
Explanation The text "This is a Type Test" was entered as an
h2 heading and then CSS was applied to change
the text color green and the font-family to monospace.		 In this example, the text "This is a Class Test" was entered as a class using a div element.
This allowed the CSS to identify the class as '.test' and only apply styling to that class.
The styling applied was an indianred text color and a bold text.		 The final example shows the text "This is an ID Test" being classified as an ID.
This specifies the element further since IDs are unique and can only be used one.
The text in the ID is colored maroon and italicized.
